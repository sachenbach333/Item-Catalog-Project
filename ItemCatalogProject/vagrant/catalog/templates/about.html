About

This Item Catalog Web Application provides a list of categories (bottles) with a variety of items within those categories and integrates third party user registration and authentication. Authenticated users should have the ability to post, edit, and delete their own items.

This application was built with Python, Flask, SQLAlchemy, HTML5, CSS, Bootstrap, and Google Oauth2 API.
